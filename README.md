# 🕵️ Python Banner Grabber

Ferramenta de identificação de serviços que realiza coleta de banners para descobrir versões e tecnologias em execução.

---

## 🎯 Objetivo

Realizar Service Fingerprinting para identificar serviços ativos e suas versões, permitindo análises de segurança mais precisas.

---

## ⚙️ Funcionalidades

- 🔎 Coleta de banners de serviços ativos.
- 🌐 Interação ativa com servidores (ex: requisição HTTP HEAD).
- 🧠 Identificação de versões e tecnologias.
- 🔄 Suporte a serviços silenciosos.
- ⚠️ Tratamento de encoding (UTF-8 / ASCII).
- 🛡️ Base para análise de vulnerabilidades (CVE).

---

## 🧰 Tecnologias Utilizadas

- Python 3.x
- socket

---

## 🚀 Como executar

Siga os passos abaixo para rodar o projeto localmente:

### 📥 1. Clonar o repositório
```bash
git clone https://github.com/emersonsilvacybersecurity/BannerGrabber.git
```

### 📂 2. Acessar a pasta do projeto
```bash
cd BannerGrabber
```

### ▶️ 3. Executar o script

#### 💻 Windows
```bash
python banner.py
```

#### 🐧 Linux
```bash
python3 banner.py
```
